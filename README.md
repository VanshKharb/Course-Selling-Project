# Course Selling Platform

A backend-driven course selling platform built with Node.js, Express, and MongoDB. This project focuses on building a scalable API for managing users, courses, and purchases with proper validation and structured architecture.

## Features

- User authentication and authorization
- Course creation and management (admin)
- Purchase system for users
- Input validation using Zod
- MongoDB database integration

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- Zod (for schema validation)
